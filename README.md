# Reverse UI React

## Quick Start

```bash
# Install dependencies
npm install

# Type-check all components
npm run typecheck

# Lint code
npm run lint

# Build (compile to JavaScript)
npm run build
```

## Structure

```
out/
├── src/                   # Components (TypeScript + JSX)
├── tsconfig.json          # TypeScript configuration
├── .eslintrc.cjs          # ESLint rules
├── package.json           # Dependencies & scripts
└── README.md              # This file

```

## Technology Stack

- **React**: ^18.0.0 (peer dependency)
- **TypeScript**: ^5.3.0
- **Emotion**: ^11.5.0 (CSS-in-JS)
- **MUI System**: ^5.15.15 (Box component, sx prop)
- **Framer Motion**: ^11.2.4 (animations)
